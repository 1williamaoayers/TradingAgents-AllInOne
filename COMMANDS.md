# 🎮 指令控制台 (Command Center)

以下是 Antigravity 系统的**核心控制指令**与**技能触发词**。
不用死记硬背，把它当成一份快捷键说明书。

## 1. 核心流程指令 (Workflows)
这些指令控制着项目的生命周期。

| 您的指令 | 对应动作 | 底层逻辑 |
| :--- | :--- | :--- |
| **"开始"** | 🚀 **开工仪式** | `start.md`<br>- 挂载记忆<br>- 检查 TODO |
| **"收工"** | 🔚 **存档协议** | `session-handover.md`<br>- 跟进任务<br>- 记录进度<br>- 写入遗言 |
| **"提交"** | 📤 **代码发布** | `deploy.md`<br>- 检查环境安全<br>- Git Commit & Push |
| **"验证"** | 🧪 **测试验证** | `test-verification.md`<br>- 运行测试<br>- 检查 Log |
| **"违规"** | 🆘 **紧急制动** | `rule.md`<br>- 停止当前操作<br>- 强制自检 |

---

## 2. 专项技能指令 (Skills)
这些指令触发特定的能力插件。

| 您的指令 | 触发技能 | 功能描述 |
| :--- | :--- | :--- |
| **"记录"** / **"记住"** | 🧠 **记忆系统** | `memory-system.md`<br>- 搜索知识库<br>- 存入长期记忆 |
| **"安装MCP"** / **"配置MCP"** | 🔌 **插件安装** | `mcp-installer`<br>- 自动修改 mcp_config.json<br>- 安装新插件 |
| **"创建技能"** | 🛠️ **技能工坊** | `skill-creator`<br>- 把重复劳动固化为 .md 文件 |
| **"监控"** / **"审查"** | 🕵️ **看门狗** | `rule-watchdog`<br>- 自动审计违规<br>- 检查语言、记忆、证据链 |

---

## 3. 常用组合技 (Combos)

*   **早晨开工**: `"开始"` -> (看汇报) -> `"验证"` (确认昨天代码没挂)
*   **开发功能**: `"记忆 搜索xxx"` (找参考) -> (写代码) -> `"验证"`
*   **下班走人**: `"提交"` (推代码) -> `"收工"` (写交接)
